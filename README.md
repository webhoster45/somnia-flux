## 🛡️ The Problem: The Block-Time Gap
In traditional DeFi, exploits happen in milliseconds. Human response (multisigs, manual pauses) take minutes or hours. By then, the vault is empty.

## ⚡ The Solution: Somnia Flux
**Somnia Flux** is a sentient security layer. It doesn't just wait for an attack—it monitors the **entire Somnia state network** for anomalies and reacts *before* the attacker can finalize their drain.

### 💎 Key Innovations
- **Neural Protection (Anomaly Detection)**: Monitors global network activity. If a massive drain (e.g., matching your custom policy threshold) occurs anywhere on the network, the firewall enters high-alert.
- **Atomic Auto-Rescue**: In a CRITICAL breach, the vault automatically midgrates 100% of its assets to a secure cold storage wallet in the same block as the detected threat.
- **Zero-Gas Reactivity**: The Somnia Reactivity Engine triggers the firewall's logic gaslessly, ensuring peak network congestion never slows down the defense.
- **Real-Time Attacker Profiling**: The system identifies the block-explorer-verified "Threat Actor" address and displays it on the command dashboard instantly.
